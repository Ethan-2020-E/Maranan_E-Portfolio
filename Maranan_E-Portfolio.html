  1. XHTML - eXtensible Hypertext Markup Language
    -> XHTML is a stricter and cleaner version of HTML

  2. Lists
    1. Ordered list
      - this type of list is where each item is numbered usually starting at 1.
      - also called numbered list
    2. Unordered List
      - a type of list that enumerates items in bulleted form without order.
    3. Description List
      - A list useful for making our own "definition of terms" or summary of each items.

  3. Tables
    - they are used to define organize data in html
    • tr -> Table rows
    • td -> Table Columns/data
    • th -> Table Headers

      Attributes for the table tag
    • align -> indicates the horizontal alignment of the table
    • background -> indicates the background image of the table
    • bgcolor -> indicates the background color of the table
    • border -> indicates the thickness of the border in pixels
    • bordercolor -> indicates the color of the border
    • cellpadding -> indicates the distance between the contents of the cells and the border around it
    • cellspacing -> indicates the distance between the cells in pixels
    • width -> indicates the width of the table

  4. Hyperlinks
    - a reference link that allows you to navigate to another page of the same document or to another document

      Types of Hyperlinks
    1. Absolute Url -> links to a page on a differet web server
    2. Relative URL -> links to a page on the same web server
    3. Named anchor -> links to a different location on the same web page.

      Attributes for the anchor tag
    • href -> indicates the target of the anchor
    • target -> indicates the behavior the web browser will carry out
    • name -> assigns a name to the anchor defining an internal bookmark of the page

  5. Forms
    - allows gathering feedback from your readers and work on the data to provide better service

      Attributes of forms
    1. Action -> responsible for indicating where the information will be passed whether it is to another web page or to a script
    2. Method -> responsible for indicating the way for sending the information

    Input -> used for creating buttons depending of the type attribute

      Values for type attribute
    1. text -> used for creating text boxes which are single line input boxes
    2. password -> for inputting passwords
    3. radio -> used for creating radio buttons which provides a variety of options but only one can be chosen
    4. checkbox -> offers a variety of options where none, one, or many may be selected
    5. submit -> submits the information when the submit button is clicked
    6. reset -> clears all input information
